Znacznik FONT

  1. Krój fonta: < font face =”nazwa_kroju” > Tekst < /font >
  2. Rozmiar fonta: < font size =”12px” > Tekst < /font >
  3. Kolor fonta: < font color =”nazwa_koloru” > Tekst < /font >
  4. Kolor tekstu: < body text= ”green” > … < /body >

Teksty - formatowanie tekstu

  1. < b >…< /b > pogrubienie
  2. < i >…< /i > kursywa
  3. < u >…< /u > podkreślenie
  4. < strike >…< /strike > przekreślone
  5. < big> …< /big > duże litery
  6. < small >…< /small > małe litery

Znacznik akapitu, atrybuty < p >:

  1. < p align=”left” >…< /p > wyrównanie do lewej
  2. < p align=”right” >…< /p > wyrównanie do prawej
  3. < p align=”center” >…< /p > wycentrowanie
  4. < p align=”justify” >…< /p > wycentrowanie
